# 🛒 TALANG SHOP Bot

Bot Discord shop system otomatis untuk server Roblox.

## Kompatibilitas
- Python 3.10.12
- discord.py 2.3.2

## Fitur
- 🎫 Open Tiket (5 kategori)
- 🛒 Showcase produk langsung di tiket
- 💳 Pembayaran QRIS otomatis
- 📦 Kirim file .rbxm otomatis setelah konfirmasi
- ⭐ Rating & Review + @everyone
- 📊 Log progres tiket (OWNER only)
- 🔔 Notifikasi @everyone
